# Yoho

## 项目说明

> 项目从 0 开始搭建，存在不合理的地方，可以自行优化~

- 技术栈：pnpm + monorepo + typescript + tsx + vitest + eslint + vuepress@next
- node 版本：>= v16.15.0；
- vue3 版本：^3.3.2；

**提交规范**

- feat: 新特性或者新功能
- fix: 修复 bug
- chore: 构建过程或者其他无关的改动
- refactor: 重构

**demo**

- 用来写测试 demo
- 后续会替换成 vuepress

**packages**

- ui
- utils
